# How to run locally:
`npm start`
Changes in code are updated after saving, no need to restart the site
Open [http://localhost:3000](http://localhost:3000) to view it in your browser.


# How to deploy:
`git add -A `
`git commit -m "COMMIT MESSAGE"`
`git push`
`npm run deploy`
